# Sistema de Diseño de Agenda de Eventos VRIN

Este archivo contiene las reglas y tokens de diseño para ser utilizados con Google Stitch.

- **Radius Small:** `8px`

- **Radius Large:** `24px`
- **Radius Full:** `9999px`

## Sombras
- **Shadow Small:** `0 1px 2px rgba(0, 0, 0, 0.05)`
- **Shadow Medium:** `0 4px 6px -1px rgba(0, 0, 0, 0.05), 0 2px 4px -2px rgba(0, 0, 0, 0.03)`
- **Shadow Large:** `0 20px 25px -5px rgba(0, 0, 0, 0.05), 0 8px 10px -6px rgba(0, 0, 0, 0.02)`
- **Shadow Hover (Primary):** `0 25px 50px -12px rgba(194, 14, 26, 0.12)`

## Reglas y Estética Principal
1. **Minimalista y Futurista:** La interfaz debe ser limpia. Se usan contenedores con bordes redondeados (`16px`) y sombras sutiles que incrementan en estados hover para añadir profundidad.
2. **Glassmorphism:** Las barras de herramientas y menús flotantes deben utilizar fondos semitransparentes con desenfoque (`backdrop-filter: blur(12px)`) para mayor inmersión.
3. **Micro-interacciones Fluidas:** Todas las animaciones deben ser suaves utilizando una transición de `all 0.4s cubic-bezier(0.16, 1, 0.3, 1)`. Los botones de iconos deben girar ligeramente o escalar en interacciones de hover.
4. **Claridad Visual:** Se le da absoluta prioridad a leer correctamente los títulos (`Inter 700+`) y al uso estratégico de los espacios negativos.
